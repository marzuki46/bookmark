# Personal Knowledge Hub AI - Product Specification

## Tujuan
Single-user Knowledge Hub dengan AI.

## Modul
- Dashboard
- Bookmark
- Notes
- Prompt Library
- Code Snippet
- File Manager
- Secret Vault

## AI
- Auto Summary
- Auto Category
- Auto Tag
- Duplicate Detection
- Collection Recommendation
- Smart Search

## Chrome Extension
Menggunakan REST API aplikasi utama dengan Personal API Token.

## Loop Engineering
1. Analisis.
2. Implementasi.
3. Test.
4. Perbaiki.
5. Test ulang.
6. Refactor.
7. Ulangi sampai semua test lolos.
8. Validasi UI, API, DB, Security.
9. Selesai hanya jika seluruh fitur memenuhi spesifikasi.
